# Arabidopsis Stress Response Analysis

## Overview
This repository contains Perl scripts and datasets for analyzing the metabolic response of Arabidopsis thaliana under various abiotic stress conditions, including cold, salt, drought, and more. The analysis is based on data from the AtGenExpress dataset and uses correlation analysis, enrichment calculations, and heatmap generation.

## Files
- **scripts/**: Contains Perl scripts used for analysis.
- **data/**: Raw data files used in the analysis.
- **results/**: Output files from the analysis, including correlation results and enrichment data.
- **heatmaps/**: Heatmap data files for visualizing the results.
